# НОУ ВИТК – Официальный сайт колледжа

Сайт **Высшего инженерно-технологического колледжа** (НОУ «ВИТК»), размещённый на GitHub Pages.  
Реализован на чистом HTML/CSS/JS без сторонних CSS-фреймворков. Поддерживает два языка: казахский и русский.

## Как запустить локально

> **Важно:** Сайт использует `fetch()` для загрузки инклюдов и Markdown-файлов.  
> Открытие `index.html` двойным щелчком **не будет работать** из-за политики CORS.  
> Необходимо запускать через локальный веб-сервер.

### Способ 1 – VS Code + Live Server (рекомендуется)

1. Установите расширение **Live Server** в VS Code
2. Откройте папку `vitk-site` в VS Code
3. Нажмите **«Go Live»** в нижней панели
4. Сайт откроется по адресу `http://127.0.0.1:5500`

### Способ 2 – Python

```bash
cd vitk-site
python3 -m http.server 8000
# Откройте http://localhost:8000
```

### Способ 3 – Node.js

```bash
npx serve vitk-site
```

## Как добавить новый раздел меню

Добавление нового раздела занимает 4 шага:

### Шаг 1. Создайте папку и HTML-страницу

Создайте `pages/news/index.html`, скопировав структуру из любой существующей страницы (например `pages/about/index.html`).

### Шаг 2. Добавьте пункт в файл `_includes/nav.html`

```html
<li class="nav-item">
  <a href="pages/news/index.html" class="nav-link" data-i18n="nav_news"></a>
</li>
```

### Шаг 3. Добавьте переводы в `assets/js/i18n.js`

В секцию `kk`:
```js
nav_news: "Жаңалықтар",
```
В секцию `ru`:
```js
nav_news: "Новости",
```

### Шаг 4. Создайте контент

Создайте `content/news/kk.md` и `content/news/ru.md` с нужным текстом.

## Как редактировать контент

Весь текстовый контент страниц хранится в папке `content/` в формате **Markdown (.md)**.

| Файл | Страница | Язык |
|------|----------|------|
| `content/about/kk.md` | О нас | Казахский |
| `content/about/ru.md` | О нас | Русский |
| `content/admissions/kk.md` | Приёмная комиссия | Казахский |
| `content/admissions/ru.md` | Приёмная комиссия | Русский |
| `content/admissions/specs.json` | Таблица специальностей | Оба языка |

Откройте нужный `.md` файл в любом текстовом редакторе и внесите изменения.  
Поддерживается стандартный синтаксис Markdown: заголовки (`#`, `##`), списки (`-`, `1.`), жирный текст (`**текст**`) и т.д.

## Как добавить фотографии в карусель

1. Поместите фотографии в папку `assets/images/`
2. Назовите их `slide1.jpg`, `slide2.jpg`, `slide3.jpg` (заменив существующие заглушки)
3. Для добавления **дополнительных слайдов** откройте `index.html` и добавьте блок:

```html
<div class="carousel-slide">
  <img src="assets/images/slide4.jpg" alt="Описание фото">
  <div class="carousel-caption">
    <h2 data-i18n="slide4_title"></h2>
    <p  data-i18n="slide4_desc"></p>
  </div>
</div>
```

Также добавьте кнопку-точку в блок `.carousel-dots`:

```html
<button class="carousel-dot" role="tab" aria-label="Слайд 4"></button>
```

И добавьте переводы в `assets/js/i18n.js` для ключей `slide4_title` и `slide4_desc`.

## Двуязычность

Система переводов построена на атрибутах `data-i18n` в HTML и файле `assets/js/i18n.js`.

- **По умолчанию** используется **казахский** язык
- Выбор языка сохраняется в `localStorage` браузера
- Переключатель **ҚАЗ / РУС** находится в правом верхнем углу шапки

### Как добавить новый текстовый ключ

1. В HTML-файле пометьте элемент: `<span data-i18n="my_key"></span>`
2. В `assets/js/i18n.js` добавьте перевод в оба раздела:

```js
kk: {
  my_key: "Казахский текст",
  // ...
},
ru: {
  my_key: "Русский текст",
  // ...
}
```

## Размещение на GitHub Pages

1. Создайте репозиторий на GitHub (например `vitk-site`)
2. Загрузите все файлы в репозиторий
3. Перейдите в **Settings → Pages**
4. Выберите ветку `main`, папку `/root`
5. Нажмите **Save**

Сайт будет доступен по адресу: `https://ваш-логин.github.io/vitk-site/`
